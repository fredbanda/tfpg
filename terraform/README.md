# My-App - Infrastructure Deployment

This directory contains Terraform configuration to deploy **my-app** to **GCP**.

## 📋 Prerequisites

1. **GCP CLI** configured with appropriate credentials
2. **Terraform** >= 1.5.0 installed
3. **Docker** image built and pushed to a registry

## 🚀 Quick Deployment

### 1. Initialize Terraform
```bash
# Run bootstrap script (AWS only - creates S3 state bucket)
./bootstrap.sh my-app

# Initialize Terraform
terraform init
```

### 2. Review and Customize
```bash
# Edit variables
nano terraform.tfvars

# Review the deployment plan
terraform plan
```

### 3. Deploy Infrastructure
```bash
terraform apply
```

### 4. Access Your Application
After deployment, Terraform will output the application URL and connection details.

## 🆘 Troubleshooting

1. **Permission Issues**: Ensure your GCP credentials have sufficient permissions
2. **Resource Conflicts**: Check for naming conflicts with existing resources
3. **State Issues**: Ensure the state backend is properly configured
